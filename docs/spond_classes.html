Introduction

Spond is a team/group-oriented events system.

The unofficial Python spond package gets data from the Spond API and returns dict objects.

This unofficial Python spond-classes package parses those dicts to Pydantic class instances.

Key classes

These are the classes intended for direct user instantiation:

  • Event via Event.from_dict()
  • Group via Group.from_dict()
